# 报告与分析脚本 / Reporting and Analysis Scripts

本目录包含用于生成各类报告的 TCL 脚本。  
This directory contains TCL scripts for generating various reports.

## 脚本列表 / Script List

| 脚本 / Script | 说明 / Description |
|---|---|
| [`report_timing.tcl`](report_timing.tcl) | 生成时序报告 / Generate timing report |
| [`report_utilization.tcl`](report_utilization.tcl) | 生成资源利用率报告 / Generate utilization report |
| [`report_power.tcl`](report_power.tcl) | 生成功耗报告 / Generate power report |
