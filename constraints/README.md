# 约束文件管理脚本 / Constraint File Management Scripts

本目录包含用于管理 XDC 约束文件的 TCL 脚本。  
This directory contains TCL scripts for managing XDC constraint files.

## 脚本列表 / Script List

| 脚本 / Script | 说明 / Description |
|---|---|
| [`add_xdc.tcl`](add_xdc.tcl) | 添加 XDC 约束文件 / Add an XDC constraint file |
| [`set_clock_constraints.tcl`](set_clock_constraints.tcl) | 设置时钟约束 / Set clock constraints |
| [`set_io_constraints.tcl`](set_io_constraints.tcl) | 设置 IO 引脚约束 / Set IO pin constraints |
