# 综合与实现脚本 / Synthesis and Implementation Scripts

本目录包含用于运行综合和实现流程的 TCL 脚本。  
This directory contains TCL scripts for running synthesis and implementation flows.

## 脚本列表 / Script List

| 脚本 / Script | 说明 / Description |
|---|---|
| [`run_synthesis.tcl`](run_synthesis.tcl) | 运行综合 / Run synthesis |
| [`run_implementation.tcl`](run_implementation.tcl) | 运行实现（布局布线）/ Run implementation (place and route) |
| [`generate_bitstream.tcl`](generate_bitstream.tcl) | 生成比特流文件 / Generate bitstream |
