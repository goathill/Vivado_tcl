# 项目创建与管理脚本 / Project Creation and Management Scripts

本目录包含用于创建和管理 Vivado 项目的 TCL 脚本。  
This directory contains TCL scripts for creating and managing Vivado projects.

## 脚本列表 / Script List

| 脚本 / Script | 说明 / Description |
|---|---|
| [`create_project.tcl`](create_project.tcl) | 创建新的 Vivado 工程 / Create a new Vivado project |
| [`add_sources.tcl`](add_sources.tcl) | 向工程中添加源文件 / Add source files to a project |
| [`export_project.tcl`](export_project.tcl) | 导出工程为可移植的脚本 / Export a project as a portable script |
