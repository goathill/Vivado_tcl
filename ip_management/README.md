# IP 核管理脚本 / IP Core Management Scripts

本目录包含用于创建、配置和管理 IP 核的 TCL 脚本。  
This directory contains TCL scripts for creating, configuring, and managing IP cores.

## 脚本列表 / Script List

| 脚本 / Script | 说明 / Description |
|---|---|
| [`create_ip.tcl`](create_ip.tcl) | 创建并配置 IP 核 / Create and configure an IP core |
| [`upgrade_ip.tcl`](upgrade_ip.tcl) | 升级工程中的 IP 核版本 / Upgrade IP cores in a project |
| [`generate_ip_output.tcl`](generate_ip_output.tcl) | 生成 IP 核输出产品 / Generate IP output products |
